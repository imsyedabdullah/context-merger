# Change Log

All notable changes to the "context-merger" extension will be documented in this file.

## [Unreleased]

## [0.0.4]

- Added file type filtering controls:
  - `contextMerger.disableFileTypeFiltering`
  - `contextMerger.additionalFileTypes`
- Added skip list setting:
  - `contextMerger.ignoredFilesFolders`
- Added content cleanup settings:
  - `contextMerger.skipExtraBlankLines`
  - `contextMerger.removeComments`
- Improved merge pipeline to support extension filtering, ignored files/folders list, and optional content cleanup before output generation.
